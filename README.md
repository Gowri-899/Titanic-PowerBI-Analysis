# 🚢 Titanic Survival Analysis (Power BI)

## 📊 Project Overview
This project analyzes the Titanic dataset using Power BI to understand survival patterns based on gender.

## 🔍 Key Insights
- Female passengers had higher survival rates
- Male passengers had lower survival rates
- Total Survived: 342
- Total Deaths: 549

## 📈 Dashboard Features
- Gender distribution (Pie Chart)
- Survival Count KPI Cards
- Survival Percentage by Gender (Bar Chart)
- Interactive Filters

## 🧠 DAX Used
- CALCULATE
- COUNTROWS
- FILTER

## 🛠️ Tools Used
- Power BI
- DAX
- Data Visualization
